HTML Kódolás Oktató Weboldal

Alapok

Mi az a HTML?

A HTML (HyperText Markup Language) egy olyan nyelv, amely segítségével weboldalakat készítünk. Nem programozási nyelv, hanem egy jelölőnyelv, amely megmondja a böngészőnek, hogyan jelenítse meg a tartalmat.

HTML elemek

A HTML oldal elemekből áll, amiket tageknek nevezünk. Minden elem általában nyitó és záró tagből áll:

<tag>Tartalom</tag>

Például:

<p>Ez egy bekezdés.</p>

A <p> az egyik leggyakoribb HTML elem, ami bekezdést jelöl.

Mi az a tag és attribútum?

A tag a HTML elem neve, pl. p, h1, img. A tag-ekhez tartozhatnak attribútumok, amik plusz információt adnak:

<tag attribútum="érték">Tartalom</tag>

Például egy kép beszúrása:

<img src="kep.jpg" alt="Leírás">

Itt a src az attribútum, ami a kép helyét mondja meg, az alt pedig a kép leírása, ami akkor látható, ha a kép nem tölt be.

Hogyan működik a weboldal?

Amikor megnyitsz egy weboldalt, a böngésző letölti a HTML kódot, majd értelmezi azt, és megjeleníti a formázott tartalmat. A HTML önmagában csak a tartalom szerkezetét határozza meg, a kinézetet a CSS szabályokkal tudjuk megadni.

Mi az a CSS?

A CSS (Cascading Style Sheets) segítségével tudjuk megmondani, hogy a HTML elemek hogyan nézzenek ki. Pl. milyen színű legyen a háttér, vagy a szöveg.

Összefoglalás

  • HTML = a weboldal szerkezete és tartalma
  • CSS = a weboldal megjelenése, formázása
  • Tag-ek = HTML elemek, amik tartalmat vagy szerkezetet jelölnek
  • Attribútumok = extra információk a tag-ekhez

Teljesen kezdőknek

1. HTML alapváz

A HTML oldal alap szerkezete meghatározza, hogyan épül fel a dokumentum, és biztosítja, hogy a böngészők helyesen jelenítsék meg az oldalt.

<!DOCTYPE html>
<html lang="hu">
  <head>
    <meta charset="UTF-8">
    <title>Oldal címe</title>
  </head>
  <body>
    Itt a tartalom.
  </body>
</html>

Magyarázat:

  • <!DOCTYPE html>: Megmondja a böngészőnek, hogy HTML5 szabványt használunk.
  • <html lang="hu">: A dokumentum nyelve magyar (hu).
  • <head>: Itt helyezkednek el az oldal meta-információi (például karakterkódolás és cím).
  • <meta charset="UTF-8">: Megadja a karakterkódolást, hogy az ékezetes karakterek helyesen jelenjenek meg.
  • <title>: Az oldal címe, ami a böngésző fülén jelenik meg.
  • <body>: Az oldal tényleges tartalma ide kerül.

2. Címsorok és bekezdések

A címsorok segítségével hierarchiát adhatsz az oldal szövegének. A legnagyobb a <h1>, a legkisebb pedig a <h6>.

<h1>Ez a főcím</h1>
<h2>Ez egy alcím</h2>
<p>Ez egy bekezdés szöveg.</p>

A bekezdéseket a <p> taggel jelöljük. A böngésző ezek között alapból térközt hagy.

3. Szövegformázás

Szövegedet különböző tagekkel formázhatod:

  • <b>: félkövér
  • <i>: dőlt
  • <u>: aláhúzott
  • <mark>: kiemelt háttér
<p>Ez a szöveg <b>félkövér</b>, <i>dőlt</i> és <u>aláhúzott</u> lesz.</p>

4. Színek megadása hex kódokkal

A színeket a CSS-ben hexadecimális kódokkal adhatod meg, pl.:

  • #FF0000 – piros
  • #00FF00 – zöld
  • #0000FF – kék
  • #000000 – fekete
  • #FFFFFF – fehér

Példa háttérszín megadására:

body {
  background-color: #e0f7fa;
}

5. Színezés CSS segítségével

Két módon színezhetsz:

  • Az egész weboldal színezése: például a body elem háttérszínének megadása.
  • Csak egy rész színezése: egy adott HTML elemnek, például egy <div>-nek vagy <p>-nek a háttérszíne vagy szövegszíne.
<!-- Egész oldal színezése -->
<style>
  body {
    background-color: #f0f0f0;
  }
</style>

<!-- Csak egy rész színezése -->
<div style="background-color: #ffccbc;">Színes doboz</div>

6. Alapvető HTML szabályok

  • Minden nyitó taghoz tartozik záró tag, kivéve az önzáró tageket (<img>, <br> stb.).
  • A tageknek helyesen kell egymásba ágyazódniuk.
  • Használj behúzásokat a kód áttekinthetősége érdekében.

Kódolás

Itt részletesen megmutatjuk, hogyan írj HTML kódot lépésről lépésre.

Egyszerű HTML oldal készítése

Írj egy egyszerű weboldalt, ami tartalmaz egy címsort és egy bekezdést:

<!DOCTYPE html>
<html lang="hu">
<head>
  <meta charset="UTF-8">
  <title>Első oldalam</title>
</head>
<body>
  <h1>Üdvözöllek!</h1>
  <p>Ez az első HTML oldalam.</p>
</body>
</html>

Magyarázat

  • <!DOCTYPE html> – megadja, hogy HTML5-öt használunk.
  • <html lang="hu"> – a dokumentum nyelve magyar.
  • <head> – meta-információkat tartalmaz (pl. karakterkódolás, cím).
  • <title> – a böngésző fülén megjelenő oldal neve.
  • <body> – a megjelenő tartalom helye.
  • <h1> – a legfontosabb címsor.
  • <p> – bekezdés, szöveg elhelyezésére.

Fontos szabályok

  • Minden nyitó taghoz tartozik záró tag (kivéve önzáró tagek, pl. <img>).
  • A tageket helyesen kell egymásba ágyazni.
  • A HTML kódot könnyen olvashatóvá tehetjük behúzásokkal.

Feladatok

Írj meg legalább 6 feladatot, amikkel gyakorolhatod a HTML alapokat!

  1. Készíts egy HTML oldalt, amely tartalmaz egy főcímet és két bekezdést!
  2. Színezd ki az egyik bekezdést CSS segítségével piros színűre a style attribútummal!
  3. Hozz létre egy <div> elemet, amelynek a háttérszíne legyen világoskék, és legyen benne egy rövid szöveg!
  4. Használj legalább három különböző HTML címsort (<h1>-től <h3>-ig) a weboldaladon!
  5. Beszúrj egy képet a weboldaladba a <img> tag segítségével, és adj neki alternatív szöveget!
  6. Írj egy egyszerű bekezdést, amelyben kiemeled a szöveg egy részét félkövér és dőlt formában!
